Scale Counselor Website

I developed a dynamic website featuring a Python-based algorithm that generates personalized meal plans tailored to user weight goals and data from a smart scale. The project integrated a Flask server to ensure fast, efficient data processing with sub-200ms response times, and utilized JSON to facilitate seamless communication between the front-end and server. I employed Bootstrap to design an intuitive and responsive HTML template, which I further enhanced with custom CSS to create interactive and user-friendly features. Additionally, I integrated a smart scale API that enabled real-time weight tracking, implementing a time-based system to monitor and manage weight loss progress effectively. To support user data storage and management, I built a secure user authentication system along with a robust database to track user information, meal plans, and progress over time. This project successfully merged fitness tracking, user data management, and interactive meal planning, providing a comprehensive tool for users to achieve their weight goals efficiently.
