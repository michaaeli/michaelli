LIDAR Coordinate Converter Project

For this project, I worked under Professor Kevin Chen at the University of Pittsburgh at the Swanson School of Engineering. My main task was creating a server that would get the local coordinates of an object (vehicle, pedestrain, object, etc.) from the LIDAR and convert it to to global WGS84 latitude and longitude coordinates, employing precise techniques such as bearing estimation and rotation angles to ensure high accuracy in the transformations.

This process required rigorous testing, during which I wrote unit tests to thoroughly validate the accuracy of the conversions and implemented Pythonâ€™s logging framework to monitor the execution of the algorithm. Additionally, I integrated YAML configurations to streamline the input of test coordinates, making the testing process more efficient. Beyond the core algorithm, I applied RESTful API design principles using JavaScript, React, and Python to facilitate real-time data extraction from LIDAR systems. This integration enabled the system to process and manage up to 10,000 coordinate points per second, ensuring both speed and reliability in handling large datasets from the LIDAR system.
